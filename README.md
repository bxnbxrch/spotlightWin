## How to run it

1. **Clone it**
   ```bash
   git clone <your-repo-url>
   cd WinSpotlight
   ```
2. **Install dependencies**
   ```bash
   npm install
   ```
3. **Run it**
   ```bash
   npm run start
   ```
4. **Build it (optional)**
   ```
   npm run build
   ```


Add this to startup for best experience.
